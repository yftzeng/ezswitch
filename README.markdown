Easy Screen Switch Mode
=======================

Install Guide
=============

System requirement
------------------

  1. Python 2.4 or later
  2. Zenity


Installation
------------

* Ubuntu

  Python and Zenity is default package in Ubuntu, so you can execute this program directly.

* Fedora

  1. Install Python

     <code>yum install python</code>

  2. Install Zenity

     <code>yum install zenity</code>
